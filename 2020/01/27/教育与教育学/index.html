教育与教育学笔记

教育与教育学

1.1教育概述

1.1.1 教育的概念

  • 广义的教育：凡是能增长人的知识和能力、影响人的思想品德、提高人的认识能力、增强人的体质、完善人的个性的一切活动
  • 狭义的教育：所谓狭义的教育，即指学校教育（狭义的教育强调教育活动的组织性、计划性、专业性）（教育学所研究的教育主要是学校教育）

1.1.2 教育的要素

  • 宏观：
    • 教育主体
    • 教育目标
    • 教育内容
    • 教育手段
    • 教育环境
    • 教育途径
  • 微观：
    • 教育者：教育者是教育过程中“教”的主体
    • 受教育者：受教育者是教育过程中“学”的主体
    • 教育内容：教育内容是师生共同认识的客体，教育内容在学校中的具体表现形式是课程标准和教科书（最佳的教育内容是目的性与对象性的统一）
    • 教育手段：教育手段是教育活动的基本条件（教育者将教育内容作用于受教育者所借助的各种形式的总和，包括物质手段、精神手段等）
      • 物质手段：可分为活动场所与设施、教育媒体、教育辅助手段三大类
      • 精神手段：包括教育方法、教育途径

1.1.3 教育的形态

  • 正规教育与非正规教育

    • 正规教育：指由教育部门认可的教育机构所提供的有目的性、有组织性、有计划性、由专职人员承担的训练和培养活动，特点是统一性、连续性、标准化和制度化。
    • 非正规教育：除正规教育之外的教育和培训活动

  • 实体教育与虚拟教育：

    • 实体教育：指在一个现实空间里的教育
    • 虚拟教育：脱离物理空间、时间限制，即以电子技术、信息技术以及网络空间为媒介而展开的教育形态

  • 社会教育、家庭教育和学校教育

    • 社会教育

      1. 社会教育的概念：

        • 广义：一切能增进人们的知识、技能、身体健康以及形成或者改变人们思想意识的活动
        • 狭义：指学校教育和家庭教育之外的一切社会文化机构以及有关的社会团体或组织对社会成员所进行的教育

      2. 社会教育的形式：

        • 勤工俭学活动
        • 社会调查实践活动等
        • 参观、访问和游览
        • 革命纪念日活动
        • 墙报和黑板报
        • 文娱体育活动

      3. 社会教育的作用：社会教育在现代化社会中的意义越来越重要，是现代社会教育体系中不可忽略的一部分，主要表现在以下四个方面

        • 社会教育直接面向全社会、又以社会政治经济为背景，比学校教育、家庭教育具有更广阔的活动余地，影响面更为广泛，更能有效地对整个社会产生积极作用。
        • 社会教育不仅面对青少年，更面对社会的成人劳动者。它不仅可以弥补学校教育的不足，满足成年人继续学习的要求，有效促进经济发展，还可以通过政治、道德教育促进社会安定与进步。
        • 社会教育形式灵活多样，没有制度化学校教育的严格约束性，能很好地体现教育的民主性。
        • 学校教育有利于个体掌握系统的知识技能，社会教育更有利于个体的社会化。

      4. 社会教育具有开放性、群众性、多样性、补偿性、融合性等特点。

    • 学校教育：学校教育在一个国家的整个教育体系中是一种主导的教育形态，学校教育系统包括各级各类学校

      • 学校教育的特点：职能的专业性、组织的严密性、作用的全面性、内容的系统性、手段的有效性、形式的稳定性
      • 局限：
      • 学校教育只占个体活动的部分时间和空间，它只是个体成长过程中特定阶段的影响要素，而不像家庭与社会伴随着个体的一生。
      • 从学校教育自身特点来看，在学校教育中学生生活被制度化（组织化）、非主体化和符号化了。

    • 家庭教育

      • 家庭教育的含义：
      • 广义：持续不断的影响
      • 狭义：有意识的教导

    • 家庭教育的性质：

      1. 家庭教育是指以家庭起源为依据的家庭规范的教育，建立在一定家庭关系基础上的教育
      2. 家庭教育是融于家庭生活中自然而然进行的教育
      3. 家庭教育是一种终身的教育过程

    • 家庭教育的意义及特点：

      • 意义：
      1. 家庭教育是人生的第一篇章，是个体社会化的最初摇篮；
      2. 家庭教育也是学习教育的重要补充；
      3. 家庭教育更能适应个体发展
      • 特点：先导性、感染性、权威性、针对性、终身性；局限在于内容的零散性与方式的随意性

    • 家庭教育、学校教育和社会教育的统合：家庭教育、社会教育、学校教育有着各自的目的、地位、作用和特点，同时它们也各自有着各自难以克服的局限性。这就需要三者之间的相互配合、协调一致，实现三种教育力量的整合，提高教育的整体效益。家庭教育和学校教育相互配合的方式和途径有互访、家长会和家长委员会。

      • 社区教育：作为整合三种教育力量的一种新形式，是一种组织体制，它具有协调育人、环境优化、组织协调、管理监督等功能,主要以学校为中心的社区教育和以社区为中心的社区教育两种组织模式。

1.1.4 教育的本质

从本质属性来说，教育就是根据一定社会需要所进行的培养人的活动，或者说是培养人的过程,教育活动是教育者和受教育者活动的统一，其中最根本的任务就是要促进受教育者身心的发展。

1.1.5 教育的基本规律

教育规律是指教育内部诸要素之间及教育内部要素与外部事物之间本质的必然联系。教育的基本规律主要有两条：一是教育与社会发展相互制约的规律;二是教育与人的发展相互制约的规律。

  • 教育与社会发展

    • 教育与社会政治经济制度之间的关系：（社会政治经济制度对教育产生制约和影响作用，社会政治经济制度决定教育的性质）
      • 社会经济制度对教育的影响
        1. 社会政治经济制度制约着教育的领导权和受教育权利
        2. 社会政治经济制度制约和影响教育目的
        3. 社会政治经济制度制约和影响教育内容
        4. 社会政治经济制度制约和影响教育制度
      • 教育对社会化政治经济制度的影响
        1. 教育能够促进年轻一代的政治社会化，培养合格公民
        2. 教育能够培养造就政治人才
        3. 教育能够传播政治意识，倡导主流政治价值观
        4. 教育能够制造社会舆论，影响政治时局
        5. 教育能够促进政治民主化
    • 教育与社会生产力的关系：主要表现为教育受生产力发展水平的制约，同时又对生产力发展起促进作用
      • 社会生产力对教育的制约作用主要表现为：
        1. 社会生产力制约教育的发展规模和速度
        2. 社会生产力制约人才培养的规格和质量
        3. 社会生产力制约课程的设置及教学内容
        4. 社会生产力影响教育结构、教育制度、教学组织方式、教学方法等变革
      • 教育对生产力发展的作用：
        1. 教育再生产劳动力
        2. 教育再生产科学技术
        3. 教育是发展科学的重要手段
    • 教育与科学技术的关系
      • 科学技术对教育具有影响作用
        1. 科学技术能改变教育者的观念
        2. 科学技术能够影响受教育者的数量和质量
        3. 科学技术能够影响教育的内容
        4. 科学技术能够影响教学手段、教学媒介等
      • 教育也对科学技术的发展起着重要作用
        1. 教育能够促进科学知识的再生产
        2. 教育能够推进科学的体制化
        3. 教育具有科学研究的功能
        4. 教育能够促进科研技术成果的开发利用
    • 教育与文化之间的关系（教育与文化是相互依存、相互制约的关系）
      1. 文化对教育具有重要影响：教育是一种特殊的文化现象，教育具有双重文化属性；教育是传递和深化文化的手段
      2. 教育反作用于文化
      3. 教育与文化相互依存、相互制约
        • 学校文化：学校全体成员或部分成员所习得且具有的思想观念和行为方式
        • 校园文化：是人们为了保证学校中教育活动顺利进行而创立和形成的一种特有的文化形态
        • 校风：是学校物质文化、制度文化和精神文化的统一体，是经过实践形成的

  • 教育与个体发展

    • 个体身心发展的概念：
      • 个体身心发展的含义：个体身心发展是指作为复杂整体的个人在从生命开始到生命结束的全部人生过程中不断发生的变化过程，特别是指个体的身心特点向积极的方面变化的过程
      • 个体的身心发展包括身体的发展和心理的发展
      • 个体身心发展的特殊性：人的身心发展是在社会实践过程中实现的；人的身心发展具有能动性
    • 个体身心发展的动因：
      • 内发论(遗传决定论)：内发论者一般强调人的身心发展的力量主要源于人自身的内在需求，身心发展的顺序也是由身心成熟机制决定的。
        1. 孟子认为人的本性是善良的；
        2. 奥地利精神分析学派创始人弗洛伊德认为人的本能是最基本的自然本能；
        3. 美国当代生物学家威尔逊把“基因复制”看成是决定人的一切行为的本质力量
        4. 美国心理学家格赛尔则强调成熟机制对人的发展的决定作用
      • 外铄论(环境决定论)：基本观点是人的发展主要依靠外在力量，诸如环境的刺激和要求、他人的影响和学校的教育等
        1. 我国古代性恶论的代表人物荀子、英国哲学家洛克都是外铄论的代表人物
        2. 美国行为主义心理学家华生甚至宣称：“给我一打健康的婴儿，不管他们祖先的状况如何，我都可以把他们培养成从领袖到小偷等各种类型的人。”
      • 多因素相互作用论(共同作用论)
